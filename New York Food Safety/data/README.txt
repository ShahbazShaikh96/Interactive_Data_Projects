DATA FOLDER

Place the dataset file here for the cleanest GitHub-ready setup:

data/nyc_inspection_data.csv

Why this folder exists:
- it gives the project a predictable local data location
- it avoids hard-coding a personal Desktop path as the primary setup
- it makes the repo easier for someone else to run

If the dataset is too large to upload to GitHub, keep it locally and
mention in the main README that the app expects the CSV at:

data/nyc_inspection_data.csv
