# Interactive Data Projects

This repository contains interactive data visualization, data wrangling, and dashboard assignments organized by project topic.

| Folder | Project Name | Main Work |
| --- | --- | --- |
| `01_Systems_Check_Learning_Goals` | Systems Check and Learning Goals | Introductory notebook with class goals and setup evidence |
| `02_Pandas_Data_Wrangling_Challenge` | Pandas Data Wrangling Challenge | Basic data loading, wrangling, and reflection on interactive visualization workflows |
| `03_Matplotlib_Scatterplot_Design_Challenge` | Matplotlib Scatterplot Design Challenge | Scatterplot critique and redesign principles for clearer visual communication |
| `04_Seaborn_Spotify_Visualization_Challenge` | Seaborn Spotify Visualization Challenge | Seaborn-based exploratory visualization work using Spotify-style music data |
| `05_Plotly_Pokemon_Visualization_Challenge` | Plotly Pokemon Visualization Challenge | Interactive Plotly visualizations and graph-object practice |
| `06_Plotly_Food_Delivery_Inclass_Presentation` | Plotly Food Delivery In-Class Presentation | Plotly Express presentation using food delivery order data |
| `07_Tidy_Data_ChessBuds_Messages` | Tidy Data Challenge: ChessBuds Messages | JSON parsing, tidying message data, and planning time-series visualization |
| `08_NYC_Food_Safety_Intelligence_Dashboard` | NYC Food Safety Intelligence Dashboard | Cleaned NYC inspection data, dashboard notebook, Dash app, and reusable data utilities |
