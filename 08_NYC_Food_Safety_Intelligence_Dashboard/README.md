# NYC Food Safety Intelligence Dashboard

Dashboard project using NYC restaurant inspection data to help users understand food safety patterns by borough, cuisine, grade, score, and location.

## Main Work

- cleans and standardizes NYC inspection records
- builds KPI metrics for citywide food safety overview
- compares average inspection scores by borough
- analyzes grade mix, monthly trends, critical violation rates, and common violation descriptions
- creates an interactive restaurant safety map
- includes a Dash app and reusable data utilities

## Files

- `NYC_Food_Safety_Milestone_3_Full_Notebook.ipynb`: full notebook version from the original repo root
- `dashboard_app/`: Dash app, project notebook, data utilities, data files, requirements, and submission README
