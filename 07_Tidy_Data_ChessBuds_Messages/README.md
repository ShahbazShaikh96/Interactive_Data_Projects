# Tidy Data Challenge: ChessBuds Messages

Data wrangling challenge using a JSON dataset of ChessBuds messages.

## Main Work

- loads nested JSON message data
- converts messages into a pandas DataFrame
- improves column readability and timestamp handling
- discusses how to structure message data for future visual analysis

## Files

- `Tidy_Data_ChessBuds_Messages.ipynb`: notebook analysis
- `chessbuds_messages.json`: source message data
